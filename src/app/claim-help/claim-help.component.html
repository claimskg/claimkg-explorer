How to use this website ?


The goal


This website is the result of the work of a french research team in cooperation with 2 research teams from Germany. Their goal was to extract data concerning claims from multiples fact-checking website such as Politifact, Snopes... Gathering these information allowed the researchers to build a knowledge base concerning these claims. This website has for goal to make it easier for journalists, psychologist or anyone to track Fake News coming from all over the globe.


The meaning of the seach tools

  • Entities

    This search bar allow you to type any entities in it. It can be human beings such as Donald Trump, Vladimir Putin for instance or non human things like White House, Mont-Blanc. If any of the things you typed is in the claim then the claim will be in the results pages.

  • Keywords

    This is where you can type some words which are being used to tag the claim.

  • Truth rating

    This criteria refers to the credibility given to the claims according to the website where it got extracted. Every website has its own ranking but in this interface only 4 types of truth rating is possible. True, Mixte when the sources are split about the truth of this claim,False and Others when we can't get any data about the truthness.

  • Author

    Here you can fill in the author of the claim you want to look at.

  • Time period

    If you click there you can select a start date and an end date which will define the time period of the claims you're looking for.

  • Languages

    You can check a box to get only the claim in a certain language.

  • Sources

    If you choose a website on this research bar you will only get for results the claim coming originaly from the fact-checking website you selected.
