What is Claims Explorer?


This is an exploratory search engine operating over ClaimsKG, a knowledge graph of fact-checked annotated claims, extracted from fact-checking web-sites.
ClaimsKG facilitates structured queries about claims, their truth values, authors, dates and other kinds of metadata.
It is generated through a (semi-)automated pipeline, which harvests data from popular fact-checking sites on a regular basis, lifts these data into an RDF/S model, which exploits established vocabularies such as schema.org and NIF, and annotates claims with related entities from DBpedia.
This is the result of the work of 2 French research teams — LIRMM (Montpellier) and LGI2P (Ales) — in cooperation with 2 research teams from Germany — GESIS (Cologne) and L3S (Hannover).
For more information, please visit https://data.gesis.org/claimskg/site/ .

Our data harvesting and publication process is in alignment with the copyright restrictions of the respective fact-checking websites.
( www.snopes.com , www.politifact.com , www.africacheck.org , www.checkyourfact.com , www.factscan.ca ...)

The meaning of the search tools


  • Entities

    This search bar allow you to type any entities in it. It can be human beings such as Donald Trump, Vladimir Putin for instance or things like White House, Mont-Blanc. If any of the things you typed appears in the claim then the claim will be in the results pages.

  • Keywords

    This is where you can type some words, which are being used to tag the claim. For example if a claim is about the US collusion probe then it might get tagged Trump.

  • Truth rating

    This criteria refers to the credibility given to the claims according to the website from where it got extracted. Every website has its own ranking but in this interface only 4 types of truth ratings are possible (taken from the normalised rating scheme of ClaimsKG): True, Mixte when the sources are split about the truth of this claim, False and Other when we can't get any data about the truthfulness.

  • Author

    Here you can fill in the author of the claims you want to look at.

  • Time period

    You can select a start date and an end date which will define the time period of the claims you're looking for.

  • Languages

    You can check a box to get only the claim in a certain language.

  • Sources

    If you choose a fact-checking website on this research bar (e.g.,Snopes) you will only get the claims coming originally from the selected website.
